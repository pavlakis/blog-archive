Zend Dojo CDN with SSL

Details
Created on Sunday, 06 March 2011 21:48

Using dojo CDN is actually much faster than pulling it from locally. That's mainly since using CDN multiple files are downloaded asynchronously.

Using Zend Framework, this is quite easily done. 

$this->dojo()->setDjConfigOption('parseOnLoad', true)
             ->setCdnBase(Zend_Dojo::CDN_BASE_GOOGLE)
             ->setCdnVersion('1.5')
             ->setDjConfigOption('usePlainJson', true);

OK. So, how about if we want to use SSL instead? In that case, we need to write a couple of lines of code. Google supports SSL for the above url; so, we'll just create a class and extend Zend_Dojo. 

class My_Dojo extends Zend_Dojo {
    const CDN_BASE_GOOGLE_SSL = 'https://ajax.googleapis.com/ajax/libs/dojo/';
}

Now, the code on the view only needs to change very slightly, to accommodate for this new change. 

$this->dojo()->setDjConfigOption('parseOnLoad', true)
             ->setCdnBase(My_Dojo::CDN_BASE_GOOGLE_SSL)
             ->setCdnVersion('1.5')
              ->setDjConfigOption('usePlainJson', true);

***Update: in order for the modules to be called using SSL, we need to specifically request them to through the djConfig var. Using ZF, we add it here: 

   ->setDjConfigOption('modulePaths', array(
                 "dojo" => GoMad_Dojo::CDN_BASE_GOOGLE_SSL ."1.5/dojo",
                 "dijit" => GoMad_Dojo::CDN_BASE_GOOGLE_SSL ."1.5/dijit",
                 "dojox" => GoMad_Dojo::CDN_BASE_GOOGLE_SSL ."1.5/dojox"
 ));

Now, let's do some tidy up. Inside the class, let's add one more method. 

    // this way we can easily change the version
    public static function getSslModulePaths($version = '1.5')
    {
        return array(
        "dojo"     => self::CDN_BASE_GOOGLE_SSL  . $version . "/dojo",
        "dijit"    => self::CDN_BASE_GOOGLE_SSL  . $version . "/dijit",
        "dojox"    => self::CDN_BASE_GOOGLE_SSL  . $version . "/dojox"
        );
    }

And the view now uses just one line instead:

->setDjConfigOption('modulePaths', My_Dojo::getSslModulePaths('1.5'));

Adding the module paths ssl directly through djConfig: http://dojo-toolkit.33424.n3.nabble.com/SSL-via-Google-CDN-td1888429.html
